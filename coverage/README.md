# Code Coverage Reports

This directory contains code coverage reports generated by [cargo-tarpaulin](https://github.com/xd009642/tarpaulin).

## Running Coverage Reports

### Using Make

```bash
make coverage
```

### Using cargo directly

```bash
cargo tarpaulin --workspace --out Html --output-dir coverage/
```

### Other Output Formats

```bash
# XML (Cobertura)
cargo tarpaulin --workspace --out Xml --output-dir coverage/

# JSON
cargo tarpaulin --workspace --out Json --output-dir coverage/

# Console output
cargo tarpaulin --workspace --out Console --output-dir coverage/

# Multiple formats at once
cargo tarpaulin --workspace --out Html --out Xml --out Json --output-dir coverage/
```

## Viewing Reports

After running the coverage command, open `coverage/tarpaulin-report.html` in a web browser.

## Ignoring Files

You can add `#[tarpaulin::skip]` attribute to functions or modules that should be excluded from coverage reports.

```rust
#[tarpaulin::skip]
fn this_function_is_not_covered() {
    // ...
}
```

## CI Integration

Add to your CI pipeline:

```bash
cargo tarpaulin --workspace --out Xml --report-dir coverage/
# Upload coverage/XML to your coverage tracking service
```
